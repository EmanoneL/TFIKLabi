Для работы с файлами существуют следующие кнопки:

  • Создать - позволяет создать файл
  • Открыть - позволяет открыть файл
  • Сохранить - позволяет сохранить файл
  • Сохранить как - позволяет сохранить файл в выбранном пользователем формате
  • Зактыть файл - позволяет закрыть файл, открытый на данный момент
  • Выход - позволяет выйти из программы

Также существуют следующие кнопки для правки текста:

  • Отменить - позволяет отменить изменение
  • Повторить - позволяет вернуться к предыдущему варианту
  • Вырезать - позволяет вырезать текст
  • Копировать - позволяет копировать информацию
  • Вставить - позволяет вставить скопированную информацию
  • Удалить - позволяет удалить информацию
  • Выделить все - позволяет выделить весь текст
